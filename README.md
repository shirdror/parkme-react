# ParkMe 🅿️

אפליקציית חניה עירונית — מציאה, דיווח ושיתוף של חניות פנויות בזמן אמת.
פרויקט Frontend (מודול 6) שנבנה ב־React + Vite, בעברית מלאה (RTL).

## הרצה מקומית

```bash
npm install
npm run dev
```

האפליקציה תעלה בכתובת שמופיעה בטרמינל (בדרך כלל http://localhost:5173).

## עמודים

| כתובת | עמוד |
| --- | --- |
| `/` | דף הבית |
| `/login` | התחברות |
| `/profile` | פרופיל |
| `/report-parking` | דיווח על חניה |
| `/parking-map` | מפת חניות |

## סטאק

- **React 19** + **React Router 7**
- **Vite** לבנייה והרצה
- **CSS Variables** למערכת עיצוב מלאה (ראה [`DESIGN.md`](DESIGN.md))
- כל הנתונים הם Dummy — ללא Backend.

מערכת העיצוב, פירוק הרכיבים ומפת הניווט מתועדים במלואם ב־[`DESIGN.md`](DESIGN.md).
